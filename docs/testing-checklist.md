# 测试清单

## 认证
- 注册
- 登录
- 错误账号密码
- 角色权限校验
- 非法角色访问商家/管理员接口
- 邮箱大小写归一化与重复注册校验
- 注册与登录请求参数校验失败返回 `422`
- 非法 token 访问受保护接口返回 `401`

## 餐厅与菜单
- 获取餐厅列表
- 获取餐厅详情
- 获取菜单
- 不可售商品处理
- 跨餐厅购物车限制
- 商家只能操作自己的餐厅和菜单
- 顾客不能访问停业餐厅的详情与菜单
- 不存在的餐厅或菜单资源返回 `404`

## 购物车
- 添加商品
- 修改数量
- 删除商品
- 重复加购同一商品时数量累加
- 新顾客首次读取购物车返回稳定空结构
- 购物车数量越界返回 `422`
- 用户不能修改或删除他人的购物车项
- 不存在的购物车项返回 `404`

## 订单
- 创建订单
- 空购物车下单失败
- 购物车商品下架后下单失败
- 查看订单列表
- 查看订单详情
- 用户不能查看他人订单
- 非法订单状态枚举返回 `422`
- 不存在的订单详情返回 `404`

## 商家端
- 创建菜单项
- 更新菜单项
- 更新订单状态
- 校验订单状态只能按合法流程流转
- 相同订单状态重复提交按无副作用成功处理
- 拒绝空白名称、非法价格等无效输入
- 已进入购物车或订单的菜品不能被直接删除
- 商家不能更新不属于自己的订单
- 不存在的菜单项和订单资源返回 `404`

## 管理端
- 查看餐厅列表
- 查看订单列表
